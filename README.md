## Environment setup (Windows instructions)
Need to make sure you're using Python 3.11 venv 
```bash
py -3.11 -m venv .venv
```

Confirm using correct version (should display Python 3.14.0)
```bash
python --version
```

Install all dependencies 
```python
pip install -r requirements.txt
```
